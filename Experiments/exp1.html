Course Description

This course introduces core concepts of programming, data structures, and web development, helping students build a solid foundation for advanced learning.

  • Data Structures and Algorithms
  • Electrical Engineering

Programming Languages

  2. JavaScript
  3. C

Course Topics

JavaScript
Basics of programming and interactivity for the web.
C++
Object-Oriented Programming for system-level applications.
